About

Medigeist is a design environment, in which you can create parametric SVG documents based on the Document Object Model (DOM) as the programming interface in your web browser.
The DOM is used as the engine to generate plain SVG which can be saved as a file ready for external procession. It runs solely on client site, no server-side scripting is required. This makes it easy to deploy and let you run Medigeist on different platforms.
A parametric SVG document can be a diagram, a process model, a technical draft, flow charts, a ground plan, an animation or just a composition of geometrical forms. The design of more complex graphical documents with regard to layout constraints and formal dependencies can be a tideous job by using common drawing applications. It's easy to draw a circle and assign style attributes but if you want to arrange a set of circles in a particular order or need to define dependencies between forms, for instance to run a sophisticated animation, your work becomes more complex and difficult to handle.
Dealing with plain SVG code is not in favour of many people, so another approach is scripting. Scripting allows you to define reusable documents, for instance a complex layout such as a process diagram with specific parameters as a template for new layouts. However, if you want to share your work in external applications or just embed it as an image on a web site, the extensive use of javascript can be an issue. Your work is less deployable with javascript, let alone security issues to be taken into account.
A solution could be to tranform the scripted work back into script-free SVG which can be processed by almost all modern SVG-capable web browsers and applications alike.
